# discord-ext-lava

A [discord.py](https://github.com/Rapptz/discord.py) extension for [lavaplayer](https://github.com/Walkyst/lavaplayer-fork)-based audio nodes.

## Installation

```shell
pip install discord-ext-lava 
```
