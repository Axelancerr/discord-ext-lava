Installation
============

Stable
------
.. tab:: Latest

    .. code-block:: shell

        pip install -U discord-ext-lava

.. tab:: Specific Version

    .. code-block:: shell

        pip install -U discord-ext-lava==0.5.1

.. tab:: Force Reinstall

    .. code-block:: shell

        pip install --force-reinstall discord-ext-lava


Development
-----------
.. tab:: Main

    .. code-block:: shell

        pip install -U git+https://github.com/aaronhnsy/discord-ext-lava.git@main

.. tab:: Branch

    .. code-block:: shell

        pip install -U git+https://github.com/aaronhnsy/discord-ext-lava.git@v0.4.x

.. tab:: Tag

    .. code-block:: shell

        pip install -U git+https://github.com/aaronhnsy/discord-ext-lava.git@v0.5.0


.. tab:: Commit

    .. code-block:: shell

        pip install -U git+https://github.com/aaronhnsy/discord-ext-lava.git@1ce8768
